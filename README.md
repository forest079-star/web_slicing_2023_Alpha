## 資料夾結構
- assets # 靜態資源放置處

  - images # 圖片放置處
  - scss # SCSS 的樣式放置處
  - layout # ejs 模板放置處

- pages # 頁面放置處

- JavaScript 程式碼可寫在 main.js 檔案

## 注意事項
- 已將 pages 資料夾內的 index.html 預設為首頁，建議不要任意修改 index.html 的檔案名稱
- .gitignore 檔案是用來忽略掉不該上傳到 GitHub 的檔案（例如 node_modules），請不要移除 .gitignore
